# Roadmap

## Phase 1

- stable event schema
- JSONL ingestion
- terminal summary
- static HTML export

## Phase 2

- serial port reader for ESP32 output
- SQLite persistence
- import/export fixtures
- richer report sections

## Phase 3

- live dashboard with filtering
- optional local web viewer
- release automation

## Non-Goals

- packet injection
- exploit execution
- active interference features
